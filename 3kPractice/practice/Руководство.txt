1. При запуске программы начинается моделирование передачи данных от источников к приёмникам.
2. Сначала создаются объекты всех устройств на основе данных из базы данных.
3. База данных должна находиться в папке с .exe файлом. (При запуске в Qt Creator - в папке build)
4. Также если нужно запустить программу только с определёнными устройствами, в эту папку добавляется файл info.txt (Для других ПМ), в котором содержится информация:
 - ID получателя
 - ID источников
 - Начальное время модели
 - Начальные радиоданные приёмника
5. В ходе работы программы логируются события синхронизации и передачи сообщения. Записи сохраняются в файл Log.txt
6. В конце работы программы (при получении сообщения приёмником), программа закрывается, а в папке с .exe файлом остаются файл Log.txt с журналом работы программы и output.txt с минимальным временем принятия сообщения.
